# Supply Chain Strategy Game

A small experimental strategy game prototype focused on production chains, settlement management, resource transformation, and eventually army building.

This project is currently in the early design and prototyping phase.

## Core Idea

The player builds and manages a settlement where every important item comes from a production chain.

Examples:

- Grain → Flour → Bread
- Ore → Iron → Weapons
- Wood → Tools → Workers
- Weapon + Person → Soldier

The main goal is to make the player feel that every unit, building, and military action depends on a functioning economy.

## Game Vision

The long-term vision is a strategy game where the player:

- builds production buildings,
- manages resources,
- feeds people and soldiers,
- transforms raw materials into useful goods,
- creates tools and weapons,
- trains units based on available equipment,
- researches faction-specific upgrades,
- trades with or attacks neighboring regions.

The game should focus more on systems, planning, and logistics than on fast reactions.
